Mentee Info Processor

This tool takes the raw CSV exports from your Qualtrics surveys— mentee_info.csv and optionally mentor_info.csv— and prepares them for direct input into your matching algorithm.

Instructions:

  1. Export your Qualtrics survey responses as CSV files and save them as mentee_info.csv (and mentor_info.csv if using).
  2. Place these CSV files in the same directory where this page is hosted.
  3. Click “Process CSV” to:
    1. Keep only required columns (Name, Email, Major, Industry, GradArea, Goal, Ethnicity, Identity, AffinityGroup, plus six weighting fields).
    2. Pass through Year if present (optional).
    3. Add a NumStudent column initialized to 0.
    4. Validate rows (names must not contain digits; weightings must be numeric).
    5. Report any missing columns or invalid values below.
    6. If no errors remain, download the cleaned CSV for your matching algorithm.
