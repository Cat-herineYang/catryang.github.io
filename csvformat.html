Mentee Info Processor

Upload your mentee_info.csv. This tool will:

  1. Keep only the required columns (Name, Email, Year, Major, Industry, GradArea, Goal, Ethnicity, Identity, AffinityGroup, and the six weighting fields).
  2. Add a NumStudent column initialized to 0.
  3. Validate each row (e.g. names contain no digits, weightings are integers).
  4. Report any missing columns or invalid values.
  5. Let you download the cleaned CSV if no errors remain.
