Mentee Info Processor

This tool takes the CSV exports from your Qualtrics surveys— mentee_info.csv and optionally mentor_info.csv— and prepares them for your matching algorithm.

Instructions:

  1. Export your survey responses from Qualtrics as CSV files named mentee_info.csv (and mentor_info.csv if used).
  2. Place those CSV files in the same folder as this page on your web server.
  3. Click “Process CSV” to:
    1. Keep only required columns (Name, Email, Major, Industry, GradArea, Goal, Ethnicity, Identity, AffinityGroup, plus six weighting fields).
    2. Pass through Year if it exists (optional).
    3. Add a NumStudent column initialized to 0.
    4. Validate each row (names must not contain digits; weightings must be numeric).
    5. Report any missing columns or invalid values below.
    6. If no errors remain, download the cleaned CSV for direct input into your matching algorithm.
