Mentee Info Processor

Upload your mentee_info.csv. This tool will:

  1. Keep only required columns (Name, Email, Major, Industry, GradArea, Goal, Ethnicity, Identity, AffinityGroup, plus six weighting fields).
  2. Year is optional.
  3. Add a NumStudent column initialized to 0.
  4. Validate each row (names contain no digits; weightings are numeric).
  5. Report missing columns or invalid values.
  6. Allow download of the cleaned CSV if no errors remain.
